# 🎵 Raaga Music

A free Spotify-style music app with songs in Tamil, Telugu, Hindi, English, Malayalam, and Kannada.

## Features
- Music player with play/pause/skip/seek
- Search songs by title, artist, or language
- Language filter pills
- Liked songs
- Playlists by language
- Now Playing screen with album art animation
- Mini player while browsing
- Mobile-first responsive design

## Deploy to Vercel via GitHub

### Step 1 — Create a GitHub repo
1. Go to [github.com](https://github.com) → **New repository**
2. Name it `raaga-music`
3. Click **Create repository**

### Step 2 — Push this code
```bash
cd raaga-music
git init
git add .
git commit -m "Initial commit — Raaga Music App"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/raaga-music.git
git push -u origin main
```

### Step 3 — Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Import your `raaga-music` GitHub repo
3. Framework: **Create React App** (auto-detected)
4. Click **Deploy**

That's it! Your app will be live at `https://raaga-music.vercel.app` 🎉

## Run Locally
```bash
npm install
npm start
```
